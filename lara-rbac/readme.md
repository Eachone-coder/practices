# Laravel 5.5 Rbac示例

### 说明

基于laravel5.5、laravel-permission-mongodb实现的rbac应用示例  
代码绝大部分参考~~复制~~ [yuansir/laravel5-rbac-example] 

[yuansir/laravel5-rbac-example]: https://github.com/yuansir/laravel5-rbac-example

### 安装

- git clone 到本地
- 执行 composer install
- 配置 .env 中数据库连接信息
- 执行 php artisan db:seed
